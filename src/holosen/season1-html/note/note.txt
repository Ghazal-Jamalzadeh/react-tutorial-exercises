Head -------------------------------------------------------------------------------------------------------------------
هیچ تگی که میخوای تو صفحه نمایش داده شود رو نباید توی تگ head بنویسی
برای کارهای خاص استفاده میشه


Title ------------------------------------------------------------------------------------------------------------------
عنوانی که توی تب بار مروزگر نمایش میده
اگر چند تا تگ تایتل رو پشت سر هم قرار بدی اولی رو در نظر میگیره

Span -------------------------------------------------------------------------------------------------------------------
مثلا میخوام توی یک تگ پی یه کلمه رو قرمز کنم
بلاک و اینلاین داریم: اسپن اینلاین است
ولی تگ اچ اینلاین نیست
ما اسپن رو توی تگ پی میندازیم اینتر و این ها هم نمیخوره توی متن
به همین خاطر بهش میگین اینلاین
ولی تگ اچ که بلاک است خود به خود اینتر میزنه

hr ---------------------------------------------------------------------------------------------------------------------
خط افقی جدا کننده میندازه

a ----------------------------------------------------------------------------------------------------------------------
بدون اتریبیوت هیچ فرقی با تگ پی ندارد

b | strong | em | i ----------------------------------------------------------------------------------------------------
هیچ فرقی با هم ندارند و صرفا اسم تگ ها متفاوت است

a --------------------------------------------------------------------------------------------------------------------
href------------------------------------
empty : refresh page
# : do not refresh - do not anything
url : open url in current screen
path : open another page
#id : navigate to id on current page ->
  آیدی ته یو آر ال صفحه قرار میگیرد. اگر این یو آر ال رو توی مرورگر بزنیم توی همونجای صفحه باز میشه
  مناسب برای طراحی منو و فایل های آموزشی یا تیترهای مختلف

target-----------------------------------
_blank	Opens the linked document in a new window or tab
_self	Opens the linked document in the same frame as it was clicked (this is default)
_parent	Opens the linked document in the parent frame
_top	Opens the linked document in the full body of the window
framename	Opens the linked document in the named iframe

برای سءوی سایت خوبه
دسترسی پذیری برای نابینایان
